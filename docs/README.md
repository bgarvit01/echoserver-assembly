# Documentation (GitHub Pages)

This folder contains the source for the project’s GitHub Pages site.

**To enable the site:** Repository → **Settings** → **Pages** → **Build and deployment** → **Source**: “Deploy from a branch” → **Branch**: `main` (or `master`) → **Folder**: `/docs` → **Save**.

The site will be available at `https://<username>.github.io/<repo>/` (or your custom domain).
